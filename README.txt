# Portfólio Pessoal – Matheus Neves (HTML + CSS)
Este projeto atende aos requisitos da atividade:
- Site funcional, com 4 links: Sobre mim, Formação, Portfólio e Contato (menu em todas as páginas).
- HTML5 (DOCTYPE) e CSS puro (sem Bootstrap, sem frameworks).
- Comentários leves explicando seções principais.
- Formulário de contato visual, sem back-end.
- Estrutura multi-página e links entre páginas (evitando depender do botão VOLTAR do navegador).

## Como editar
- Edite os textos em cada arquivo .html.
- Troque os links "#" da página `portfolio.html` pelos seus repositórios/deploys.
- Ajuste cores no `styles.css` (variáveis CSS em :root).
- Dica: mantenha o cabeçalho/menu idêntico em todas as páginas.

## Como publicar (opção: GitHub Pages)
1) Crie um repositório: `matheus-neves-portfolio` (ou similar com seu nome).
2) Envie estes arquivos para o repositório.
3) No GitHub, em Settings → Pages → Build and deployment → Configure para "Deploy from branch" e selecione a branch principal (geralmente `main`) e a pasta `/ (root)`.
4) Aguarde o endereço `https://seu-usuario.github.io/matheus-neves-portfolio/` ficar disponível.
5) Teste todos os links antes de entregar.

## Como publicar (opção alternativa: Netlify ou Vercel)
- Crie uma conta gratuita, arraste e solte a pasta do projeto para fazer o deploy estático.
- Use seu nome no subdomínio (ex.: `matheus-neves.netlify.app`).

## Entrega
- Envie o link publicado e o código conforme o modelo de documento da disciplina.
- Recomendado exportar em PDF para evitar desconfigurações.
